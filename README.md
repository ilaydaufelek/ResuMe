# Resume Builder

A modern and minimal resume builder built with Next.js, TypeScript, and React.  
Designed to be simple, fast, and practical.

---

## Features

- No authentication required
- Clean and modern resume template
- Real-time preview while editing
- Local storage support
- Responsive layout (desktop & mobile)
- Component-based and easy to extend

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod

---

## Usage

- Fill in your resume information
- See changes instantly in the preview
- Data is stored locally in the browser

---

## Project Status

Actively maintained and open for improvements.
